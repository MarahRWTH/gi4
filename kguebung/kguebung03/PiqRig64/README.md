# PiqRiq

Bildverarbeitung mit Assembler

## 1. Implementiere fehlenden Code

- [ConvertToBlackWhite.asm](ConvertToBlackWhite.asm)
- [DrawDiagonale.asm](DrawDiagonale.asm)
- [Invert.asm](Invert.asm)

## 2. Compile

```bash
qmake
make
```

## 3. Ausführen

```bash
./PiqRiq Linux_logo.jpg
```
